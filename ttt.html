Tic-Tac-Toe

A classic 2 person Tic-Tac-Toe game implementation using C++

Challenges Faced

  • validating incorrect user inputs
  • creating an algorithm that can handle win conditions, including ties

Technologies Used

  • C++ for game structure and logic
  • g++ complier for code execution and debugging

Outcome:

Created a functional and visually appealing Tic-Tac-Toe game that highlights interactive design, modular coding practices, and player feedback when invalid inputs are received.
